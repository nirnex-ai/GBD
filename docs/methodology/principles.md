# Core Principles

## Fact

These principles reduce hallucination risk and make AI output auditable.

## Opinion

They will feel heavy to teams that prefer fast, loose prototyping.

## Principles

1. Design is the source of expected truth.
2. Code is the source of actual truth.
3. The gap is the only valid implementation unit.
4. No implementation without evidence.
5. No gap is complete without proof.
6. Parallelism is allowed only for non-conflicting gaps.
7. The gap ledger is the project control plane.
8. AI proposes and implements; humans approve boundaries and release.
